# LUNG-MRI-WAVEFORM-EXTRACTION

YEAR 3 PROJECT

PHAS0052 Project Brief 

Supervisor: Dr Mina Kim, Principal Research Fellow (mina.kim@ucl.ac.uk) 

Project Title:  
Comparing Practical Methods for Respiratory Waveform Extraction from Lung MRI 
Project Description:  
A student group will compare three methods for generating a respiratory waveform (a breathing 
signal) from a time-series of lung MRI images. A trained segmentation model and co-registration 
outputs will be provided. Students will not develop ML models or registration algorithms, but will 
perform the analysis steps needed to produce waveforms, quantify agreement, and evaluate 
robustness and usability.  
Why this matters:  
Respiratory waveform extraction is a key preprocessing step in quantitative lung MRI. Several 
methods are used in practice, but their performance has not been systematically compared 
under consistent conditions. This project addresses that gap and provides career-relevant 
experience in imaging post-processing, with potential for a methods-style publication. 
Project Goals: 
• Apply three methods (manual vs automated intensity-based vs deformation-based) to 
extract respiratory waveforms from dynamic lung MRI data 
• Quantitatively compare the methods in terms of agreement, robustness, and usability 
• Assess how analysis choices affect the resulting respiratory signal 
• Develop a reproducible analysis pipeline using MATLAB or Python 
• Work effectively as a team and communicate results clearly through figures, reports, and 
presentations 
Dataset and Inputs Provided: 
• Dynamic 2D multi-slice lung MRI time-series 
• A pre-trained lung segmentation model 
• Co-registered outputs including Jacobian maps from non-rigid registration 
• Example scripts (MATLAB/Python) can be provided to reduce overhead 
Outcomes:  
By project end, students will produce:  
• A working comparison pipeline for respiratory waveform extraction 
• Evidence-based guidance on which method is preferable under which constraints 
• A set of figures/tables that could be the basis of a short methods manuscript (subject to 
results quality and novelty)
